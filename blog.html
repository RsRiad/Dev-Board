What are the different ways to select an element in the DOM?
  • getElementById("id")
  • getElementsByClassName("class")
  • getElementsByTagName("tag")
  • querySelector("selector")
  • querySelectorAll("selector")
What is the difference between innerHTML, innerText, and textContent ?
  • innerHTML – Retrieves or sets the HTML content inside an element, including HTML tags.
  • innerText – Retrieves or sets only the visible text inside an element, ignoring hidden elements and formatting.
  • textContent – Retrieves or sets all text inside an element, including hidden elements but without interpreting HTML.
What is event delegation in the DOM?
Event delegation is a technique where a parent element listens for events on its child elements instead of attaching event listeners to individual children. It works by using event bubbling.
What is event bubbling in the DOM?
Event bubbling means that when you click on a child element, the event first happens on that element and then moves up to its parent and other ancestors.
